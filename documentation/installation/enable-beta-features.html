    • Enable Beta Features

    Select Labs from the navigation menu on the left of your admin area.

    Then scroll down and at the bottom you will see an option which says Beta features.

    Public API


    Public API It is required for the proper working on this theme.

    For full instructions, read the official developer guide

    Without enabling Public API theme will not work properly

    Subscribers


    The current version of subscribers does not send emails . The Subscribers feature allows you to capture email addresses on your blog, which can then be exported and imported.

    More info in the official docs

    Without enabling Subscribers widget newsletter will be disabled
