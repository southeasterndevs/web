    How to Add Any Language

    Open ../locales folder. Copy and paste en.json, change your lanquage code (e.g.: es for Spanish, fr for French, pl for Polish). The language_code must be replaced with a valid code from.

    Translate the included sentences

    Edit the renamed translation file with any plain text editor.
    As you can see, it’s usually just plain English on the left, and the language you choose on the right.

    Activate the translation file

    Go to the admin panel and select General from the navigation menu on the left of your admin area.

    Activate your language code and click on the Save settings button.

    Congratulation, your blog is translated!

    Reactivate the translation file

    If you edited the active language, you have to restart Ghost to make the changes effective.
    Alternatively, go to the admin panel and select General from the navigation menu on the left of your admin area.

    Activate another language code and click on the Save settings button, then switch back to your language code.

    Alternatively, go to the admin panel, activate another theme, then switch back.

    Translation support in this theme requires Ghost version 1.22.0 or later
