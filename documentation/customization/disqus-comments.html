    • Disqus Comments

    After you’ve added your site to Disqus, you will be asked to choose a platform.
    Select Universal Code from the list of options.

    Once you’ve selected Universal Code, the next screen will present you with a Disqus script that you will need to copy to your Ghost blog.

    Now that you have your Universal Code copied, you’ll need to locate and open the theme’s ../partials/disqus.hbs file in your HTML editor.

    Paste your Universal Code and save your disqus.hbs file in ../partials/ folder.

    You should now have a Disqus comment box

    Disable disqus on a particular subpage

    If you don’t want to use Disqus on a particular subpage, you can turn it off by using the tag #disqus-off
